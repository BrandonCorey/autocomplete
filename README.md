## Autocomplete
Implemented autocomplete functionality from scratch
- Enter a country name to see it in action

Features:
- Tab completion
- Arrow down/up
- Enter
- Escape

## Install
### 1. `git clone <reponame>`

### 2. `npm install`

### 3. `npm start`
